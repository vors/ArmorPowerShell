    New-ArmorApiToken¶

    SYNOPSIS¶

    Retrieves an authentication token.

    SYNTAX¶

    New-ArmorApiToken [-Code] <String> [[-GrantType] <String>] [[-ApiVersion] <String>] [<CommonParameters>]

    DESCRIPTION¶

    Retrieves an authentication token from a temporary authorization code.

    EXAMPLES¶

    EXAMPLE 1¶

    New-ArmorApiToken -Code '+8oaKtcO9kuVbjUXlfnlHCY3HmXXCidHjzOBGwr+iTo='

    Submits the temporary authorization code to retrieve a new Armor API session token.

    PARAMETERS¶

    -ApiVersion¶

    Specifies the API version for this request.

    Type: String
Parameter Sets: (All)
Aliases:

Position: 3

Accept wildcard characters: False

    -Code¶

    -GrantType¶

    Specifies the type of permission.

    Type: String
Parameter Sets: (All)
